Clona el repositorio en tu maquina local:

`git clone git@github.com:geraldbustosm/trenes.git`

`git checkout -b "H001-PRTestFelipeRuiz"`

`git add .`

(Puedes hacer todos los commit que te parezcan correctos en esta rama)
